# Vision Backbones

This folder is reserved for local visual backbone code, Torch Hub cache files, and optional checkpoints.

Recommended layout:

```text
vision_backbones/
  dinov2/       # Optional local clone of https://github.com/facebookresearch/dinov2
  resnet/       # Optional local ResNet checkpoints, e.g. resnet50.pth
  torch_hub/    # Optional Torch Hub cache/checkpoints
```

Large model files are intentionally ignored by Git. Keep only placeholders and this README in the repository.
